## 🏓 What is PingPong?

**PingPong** is an automated grid trading platform built on the **BASE blockchain**. It connects directly to your Bitvavo or Coinbase exchange account via API and places intelligent buy/sell orders across a price range — earning profit from natural market volatility, 24/7.

Think of it as a **ping pong game for your money**: buy low, sell high, automatically, over and over again.

## 🔒 Security

- **Non-custodial** — your API keys stay on your own server
- **No withdrawal permissions** — trading API keys only (View + Trade)
- **Server token authentication** — all API calls require a secret token
- **Open source contracts** — all smart contracts verified and public
- **No private keys** — PingPong never touches your wallet private keys
